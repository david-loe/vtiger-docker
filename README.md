# vtiger-docker
vtiger crm Dockerized

## Installation

1. run `docker-compose up -d`
2. goto `localhost:8080`

## Settings

### PHP
change settings in `php.ini`

### Database
| Entity        | Value     |
|---------------|-----------|
| Host name     | mysql     |
| User name     | root      |
| Password      | pw        |
| Root User     | root      |
| Root Password | pw        |
| Database Name | vtiger_db |
